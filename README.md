# gimmickshelter
Independent rock webzine built with WordPress, featuring articles, reviews and curated content around alternative, garage and underground music culture.

🔗 https://www.gimmickshelter.fr/

## 🎸 Project Overview

This project is a custom WordPress theme powering a music-focused editorial platform.

The website features:
- Articles and opinion pieces
- Music reviews
- Editorial content around rock culture
- A curated selection of underground and alternative artists

The goal is to create a raw, authentic and independent media space inspired by DIY culture.

## 🛠️ Tech Stack

- WordPress (PHP)
- HTML / CSS / JavaScript
- Custom theme development
