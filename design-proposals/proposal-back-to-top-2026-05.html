Animation d'apparition (commune aux 3 variantes)

Entrée au scroll

opacity 0→1 + translateY(12px→0)
Easing : cubic-bezier(.22,.68,0,1.2)

Sortie en haut de page

opacity 1→0 + translateY(0→8px)
Durée : 250ms — ease-in

prefers-reduced-motion

Aucune animation. Apparition / disparition instantanée via opacity seulement. Smooth scroll désactivé → saut direct.

Corrections techniques (toutes variantes)

JS — Affichage

  • Supprimer window.onscroll dans main.js — conflit avec l'event passif du header
  • Utiliser opacity + pointer-events au lieu de display:none/block → transition CSS fonctionnelle
  • Seuil d'apparition : 300px de scroll (vs 20px actuel, trop tôt)
  • Merge dans le listener scroll existant du header GS

JS — Scroll

  • Remplacer scrollTop = 0 par window.scrollTo({ top: 0, behavior: 'smooth' })
  • Désactiver smooth sous prefers-reduced-motion : behavior: 'instant'
  • Supprimer onclick="topFunction()" inline dans le PHP
  • Attacher l'event via addEventListener('click') dans main.js

CSS — Nettoyage

  • Retirer les 3 !important sur background, border, color → pas de conflit à résoudre
  • Retirer outline: none → remplacer par :focus-visible avec outline blanc
  • Utiliser opacity: 0/1 + pointer-events: none/auto pour l'affichage
  • Transition : opacity .35s, transform .35s — ne jamais transition: all

HTML / PHP

  • Supprimer style="display:none;" inline → géré en CSS (opacity:0; pointer-events:none)
  • Garder aria-label="Retour en haut de page" ✅
  • Remplacer <i class="fas fa-chevron-up"> par SVG inline (plus de dépendance Font Awesome)
  • Garder dans parts/progress-bar.php ✅
Recommandation — Variante B Le cercle dark + liseré rouge est la solution la plus cohérente avec le design system : discret sans être invisible, il utilise les deux couleurs signature (noir + rouge) et s'efface derrière le contenu au repos. Le hover rouge full est une réponse claire à l'interaction. La variante A est plus assertive (bon si on veut un CTA fort) ; la variante C est plus typée "webzine print" mais peut sembler encombrante sur mobile.
