Tokens utilisés — Header

  • Bandeau logo : fond --color1 #E81F1F · texte --color3 #FFFFFF
  • Barre nav : fond --color4 #000000 → scroll : --color2 #000B29
  • Liseré scroll : --color1 #E81F1F 3px border-bottom
  • Hover liens nav : soulignement --color1 animé scaleX(0→1) depuis la gauche
  • Overlay mobile : fond --color4 #000000 · slide-in translateX depuis la droite
  • Liens overlay : Anton 2rem · hover → --color1 + translateX(4px)
  • Croix fermeture : rotation 90° au hover
  • Typos : Anton (logo, liens mobile) · Roboto (nav desktop, recherche)

Search — Loupe nav → overlay pleine largeur

  • Au repos : icône loupe fa-search à droite des liens, opacité 70% · discrète
  • Au clic : la nav disparaît (fade out 250ms) · un overlay pleine largeur apparaît (fade in 250ms ease-out)
  • Fond barre : reste --color4 #000000 (ou --color2 #000B29 si scrollé) — pas de changement de fond
  • Loupe fixe à gauche : --color1 #E81F1F
  • Input : pleine largeur, pas de border, pas de fond — texte blanc · placeholder --color5 #6D6D64
  • Au focus input : border-bottom --color1 apparaît sur la barre nav (classe search-focus)
  • Croix ✕ à droite : fermeture + reset · hover → --color1 + rotation 90°
  • Escape : fermeture aussi · focus retour sur le bouton loupe
  • Mobile : même comportement — loupe visible à côté du hamburger dans la barre noire
