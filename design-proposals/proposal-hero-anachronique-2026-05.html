Notes de conception · Hero v3

Champs ACF

  • image_hero → fond flouté du hero (photo concert, ambiance…) nouveau
  • card-square → pochette album, nette en avant-plan wp natif
  • labels → artiste rouge capitales existe
  • the_title() → titre Anton XXL wp natif
  • genre → pills ghost existe
  • annee_sortie → pill année de sortie nouveau
  • accroche → tagline italic filet rouge (desktop only) nouveau
  • get_the_author() + get_the_date() → byline wp natif

Choix visuels

  • Fond : lrg_size — blur(10px) + grain noise 4% + double gradient
  • Pochette card-square nette en avant-plan, pas floutée
  • Filet rouge var(--color1) en bas du hero
  • Mobile : pochette 100×100 flottante haut-droite, accroche masquée
  • Hauteur desktop : min(85vh, 680px)
  • Pas de note — retirée

Nouveaux champs (specs backend)

  • image_hero — type image · "Image de fond (hero)" · non requis · return : URL · fallback : thumbnail WP si vide
  • annee_sortie — type text · "Année de sortie" · non requis · ex. 2013
  • accroche — type textarea · max 160 car. · non requis · "Phrase éditoriale courte pour le hero"

Responsive

  • Desktop ≥ 641px : grille 260px + 1fr, pochette à gauche
  • Tablette 641–900px : grille 200px + 1fr
  • Mobile ≤ 640px : colonne unique, pochette 100px flottante
  • Accroche visible desktop uniquement
  • Genre pills : 2 max sur mobile pour éviter le débordement
